# v0.2.0 Release Checklist

## 1. 版本门禁

发布前确认：

- `CHANGELOG.md` 已同步
- `README.md` 已同步当前能力与运行方式
- `docs/roadmap.md`、`docs/progress/`、`docs/journal/` 已同步
- `docs/qa/` 中测试与验证资料完整
- GitHub `main` 分支处于可构建状态

## 2. 技术门禁

- `cargo fmt --all`
- `cargo clippy --workspace --all-targets -- -D warnings`
- `cargo test --workspace`
- `bash scripts/smoke_v0_2_0.sh`

## 3. 业务与治理门禁

- 首条 pilot workflow 规格已冻结
- 审批责任边界已文档化
- SOP 影响范围已说明
- 回退与人工接管策略已说明
- 审计和数据保留边界已说明

## 4. 发布动作

1. 确认版本号
2. 提交最终文档和代码
3. 打 `v0.2.0` tag
4. push tag 到 GitHub
5. 检查 release workflow 结果
6. 归档发布说明和风险结论

## 5. 发布后检查

- Release artifact 可下载
- 版本说明可阅读
- 回退方案可执行
- 已知风险已登记
